GeradorDeClientes
=================

Aplicação ASP.NET Core (Razor Pages) que gera planilhas XLSX com dados fictícios e permite envio por e-mail.

Como rodar?
-------------------
- Restaurar e compilar:

`powershell
dotnet restore
dotnet build
`

- Executar:

`powershell
dotnet run --project .\GeradorDeClientes.csproj
